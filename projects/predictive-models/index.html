Statistical and machine learning models

Decision Trees

A from-scratch implementation of the ID3 algorithm — pick splits by information gain, grow to depth 2, predict by majority vote at each leaf — alongside a comparison against sklearn's DecisionTreeClassifier at depths 2 through 5. Built for my Predictive Analytics (ECON 475) class.

The interactive demo lets you choose depth, criterion (entropy vs. Gini), and a random seed for the train/test split, then renders the actual tree alongside live train/test accuracy.

Open Decision Trees

Nearest Neighbors

A from-scratch k-NN classifier with selectable distance metric (Euclidean, Manhattan, Minkowski-3, cosine), a sklearn benchmark comparing kd_tree vs brute backends, and a synthetic dataset where cosine similarity meaningfully outperforms magnitude-based distances.

The interactive demo lets you tune k and the metric, then shows the test-set accuracy and a sample of predictions against ground truth.

Open Nearest Neighbors

Naive Bayes

Both the full-joint Bayesian MAP classifier and the Naive Bayes approximation, with Laplace smoothing for discrete features and a Gaussian likelihood for continuous ones. The independence assumption is almost always wrong — and the classifier almost always works anyway.

The interactive demo lets you flip between discrete and continuous feature sets, tune Laplace smoothing strength, and inspect the per-class posterior probability bars for any row in the dataset.

Open Naive Bayes

Gradient Descent

Batch gradient descent for linear and logistic regression, written from scratch — MSE for linear, binary cross-entropy with a numerically stable sigmoid for logistic, plus a safety brake that halts if the loss bumps upward. The "extra credit" path standardizes the feature so descent converges from a random initialization in under 1,000 steps.

The interactive demo fits the live dataset, plots the loss curve dropping step-by-step on a canvas, and lets you tune the learning rate, standardization, and mode in real time.

Open Gradient Descent

AI-Augmented Stepwise

My Generative AI for Business and Data Analysis (IS 303) + ECON 475 final-project crossover: a head-to-head between three forward-stepwise regressions on California Housing. A baseline on 8 raw features, an AI-augmented version where Claude proposes 10 engineered features given only the column names and target, and a kitchen-sink expansion with every pairwise product and square. The question: can an LLM's domain knowledge match brute-force search?

The interactive walkthrough steps you through the exact prompt sent to Claude, the JSON response that came back, each generated feature with its rationale, and the final RMSE comparison across all three models.
