How signal becomes adjustment

        
From Section 7.2 of the system documentation:

        
    
          
  1. Pattern extraction — derive features (location, tenure pattern, skills, certs) present in successful vs. unsuccessful hires.
    2. 
          
  2. Weight delta calculation — features correlated with positive outcomes shift up; failures shift their features down.
    3. 
          
  3. Confidence gating — adjustments only apply once a feature has at least 10 outcomes behind it. The first hire produces no adjustment; by the fiftieth, the model has meaningful signal.

Stage 5b · Recruiter disagreement signal

Stage 6 · Per-company calibration profile

        
Each company carries an accumulated profile: explicit rules they set directly, plus learned feature weights derived from outcomes and overrides. Weights are clipped to [0.2, 2.5] so no single feature can dominate.
