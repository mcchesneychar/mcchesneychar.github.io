Resume-to-Job Matching System

This interactive viewer is the static, read-only front end for a six-stage resume-to-job matching system I built as the final project for my Generative AI for Business and Data Analysis (IS 303) class. It ranks candidates against job postings, then layers on a per-company calibration profile that learns from hiring outcomes and recruiter overrides over time.

The demo surfaces every artifact the pipeline produces — top matches per candidate, the parsed schemas for each ingested document, the feedback loop that turns hires and rejections into feature-weight adjustments, the audit log of recruiter disagreements, and each company's accumulated calibration profile.

This was my first time combining document parsing, semantic ranking, and a learned per-entity preference layer in one pipeline — and the first project where I had to design the audit trail and explainability surface alongside the model itself.

Ramen Ratings Dashboard

This interactive dashboard contains 4 different visualizations displaying different metrics based on style, ratings, reviews, and country.

As part of my course work for my Generative AI for Business and Data Analysis class, this was my first experience using a coding agent (outside of the work I have done using it for the development of this site). It utilizes a CSV file containing 2500+ entries of brands, variety, style, country, and rating.

By utilizing proper contract and prompt writing, as well as navigating challenges such as embedding a CSV file to ensure independent access, it was a great first experience using coding agents.
